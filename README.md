## Localization File Format

The localization files are in JSON format. Each language should have its own JSON file.

**Example (en.json):**

```json
{
  "app_title": "KCD2 Tweakmods",
  "file_menu": "File",
  "open_file": "Open File",
  "save_file": "Save File",
  "exit": "Exit",
  "edit_menu": "Edit",
}
